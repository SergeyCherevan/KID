# Промпты для генерации иллюстраций к Уроку 1

Используйте эти промпты в генеративных моделях (DALL·E, Midjourney, Stable Diffusion и т.п.) для создания иллюстраций к уроку.

---

## kid-ide-main.png

**Промпт (русский):**  
Детский редактор кода .KID IDE: современный интерфейс с тёмной или светлой темой, слева — дерево файлов проекта, в центре — большое окно редактора кода с подсветкой синтаксиса C#, справа — панель с кнопкой «Запуск» (Play). Минималистичный, дружелюбный дизайн для детей. Стиль: плоский, яркие акценты, крупные иконки.

**Промпт (English, для лучшей совместимости):**  
Children's code editor .KID IDE: modern interface with dark or light theme, file tree on the left, large code editor window with C# syntax highlighting in the center, Run/Play button on the right. Minimalist, kid-friendly design. Flat style, bright accents, large icons. Educational software UI.

## hello-world-result.png

**Промпт (русский):**  
Окно программы: внизу — консоль с текстом «Hello World!», сверху — графическое окно 300x300 пикселей. Красный круг, внутри него синий квадрат, в центре белый текст «Hello / World!» в две строки. Чистый, детский стиль, яркие цвета.

**Промпт (English):**  
Program output window: console with "Hello World!" text at the bottom, graphics window 300x300 pixels above. Red circle, blue square inside, white text "Hello World!" in two lines in the center. Clean, child-friendly style, bright colors. Simple geometric shapes.
