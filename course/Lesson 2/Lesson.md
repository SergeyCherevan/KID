## 2. Продолжаем разбираться с графикой

### 2.1. Новые фигуры: линии, точки, эллипсы

Помимо круга и прямоугольника, в .KID есть ещё несколько «кирпичиков», из которых удобно собирать рисунки.

#### Линия

**Линия** — это отрезок между двумя точками. Команда:

- `Graphics.Line(x1, y1, x2, y2)` — рисует линию от точки (x1, y1) до точки (x2, y2).

Цвет линии задаётся так же, как и у других фигур: перед рисованием напиши `Graphics.Color = "..."`.

#### Точка

Отдельной команды «нарисовать точку» в .KID нет. Но точку легко сделать из **очень маленького круга**: достаточно взять радиус 1 или 2. Например:

- `Graphics.Circle(100, 100, 2);` — это и есть «точка» в центре экрана.

Такими точками можно отмечать координаты, рисовать звёзды или пунктирные узоры.

#### Эллипс

**Эллипс** — это «вытянутый круг»: у него два радиуса — по горизонтали и по вертикали.

- `Graphics.Ellipse(x, y, radiusX, radiusY)` — рисует эллипс с центром в (x, y); radiusX — полуширина, radiusY — полувысота.

Если сделать radiusX и radiusY одинаковыми, получится обычный круг — как от `Graphics.Circle`. А если разные — получится овал.

Открой и запусти файл `01_LinesPointsEllipses.cs`:

```csharp
using System;
using KID;

Console.WriteLine("Линии, точки и эллипсы");

// Линия от (50, 50) до (250, 50)
Graphics.Color = "DarkBlue";
Graphics.Line(50, 50, 250, 50);

// Несколько «точек» — маленьких кругов
Graphics.Color = "Red";
Graphics.Circle(80, 120, 2);
Graphics.Circle(150, 120, 2);
Graphics.Circle(220, 120, 2);

// Эллипс (овал)
Graphics.Color = "ForestGreen";
Graphics.Ellipse(150, 200, 80, 40);

// Круг для сравнения (та же «высота», но равные радиусы)
Graphics.Color = "Gray";
Graphics.Circle(150, 280, 40);
```

На холсте появятся горизонтальная линия, три красные точки и зелёный овал, а под ним — круг. Так ты видишь разницу между эллипсом и кругом.

### 2.2. Несколько способов задать цвет

В первом уроке мы задавали цвет **названием**: `"Red"`, `"Blue"`, `"White"`. Это удобно и понятно. Но в программировании и в графических редакторах цвета часто задают **числами**. Так компьютер точно знает оттенок.

#### 1) Название цвета (уже знакомое)

- `Graphics.Color = "Red";`
- `Graphics.Color = "DodgerBlue";`

Список названий большой: можно пробовать `"Coral"`, `"Gold"`, `"ForestGreen"` и т.д.

#### 2) Кортеж (R, G, B)

Цвет на экране получается из трёх «лампочек»: **R** (красный), **G** (зелёный), **B** (синий). Каждая «лампочка» может гореть от 0 до 255.

- 0 — не светится, 255 — светится на полную.

Примеры:

- `Graphics.Color = ((byte)255, (byte)0, (byte)0);` — чистый красный (R=255, G=0, B=0).
- `Graphics.Color = ((byte)0, (byte)128, (byte)255);` — голубой (немного зелёного, много синего).

Числа в скобках — это и есть кортеж из трёх компонент цвета.

#### 3) HEX-значение

В интернете и в редакторах часто пишут цвет в виде **шестнадцатеричного кода**:

- В коде можно написать число: `0xRRGGBB`, например `0xFF0000` — красный, `0x0000FF` — синий.
- Или строку: `"#FF0000"`, `"#F00"` (короткая форма) — .KID понимает и такие записи.

Примеры:

- `Graphics.Color = 0xFF0000;` — красный.
- `Graphics.Color = "#00FF00";` — зелёный.
- `Graphics.Color = "#F0F";` — короткая запись (малиновый).

Открой и запусти файл `02_ColorsFormats.cs`:

```csharp
using System;
using KID;

Console.WriteLine("Три способа задать цвет");

// 1) Название
Graphics.Color = "Coral";
Graphics.Circle(80, 80, 30);

// 2) Кортеж (R, G, B) — числа от 0 до 255
Graphics.Color = ((byte)255, (byte)165, (byte)0);   // оранжевый
Graphics.Circle(160, 80, 30);

Graphics.Color = ((byte)100, (byte)149, (byte)237);  // голубой
Graphics.Circle(240, 80, 30);

// 3) HEX: число 0xRRGGBB или строка "#RRGGBB"
Graphics.Color = 0x9370DB;   // фиолетовый
Graphics.Rectangle(50, 150, 80, 50);

Graphics.Color = "#20B2AA";  // морской волны
Graphics.Rectangle(150, 150, 80, 50);

Graphics.Color = "#FFD700";  // золотой
Graphics.Rectangle(250, 150, 80, 50);
```

На холсте появятся круги и прямоугольники разных оттенков — все заданы разными способами.

### 2.3. Как задать шрифт текста

Надпись на холсте можно сделать **крупнее** или **другого начертания**. Для этого перед командой `Graphics.Text(...)` нужно задать шрифт:

- `Graphics.SetFont("ИмяШрифта", размер);`

Размер — это число (например, 24 или 36). Имя шрифта — как в системе: `"Arial"`, `"Times New Roman"`, `"Segoe UI"` и т.д.

Открой и запусти файл `03_FontDemo.cs`:

```csharp
using System;
using KID;

Console.WriteLine("Разные шрифты и размеры");

Graphics.Color = "Black";

Graphics.SetFont("Arial", 16);
Graphics.Text(20, 30, "Маленький Arial 16");

Graphics.SetFont("Arial", 28);
Graphics.Text(20, 70, "Крупный Arial 28");

Graphics.SetFont("Times New Roman", 22);
Graphics.Text(20, 120, "Times New Roman 22");

Graphics.SetFont("Segoe UI", 24);
Graphics.Text(20, 170, "Segoe UI — привет!");
```

Ты увидишь одну и ту же надпись в разном размере и в разных шрифтах. После `SetFont` все следующие вызовы `Graphics.Text` будут использовать этот шрифт, пока ты не вызовешь `SetFont` снова.

### 2.4. Для самых любознательных: как рисовать многоугольники

Многоугольник — это фигура из нескольких вершин, соединённых отрезками (треугольник, пятиугольник, звезда и т.д.). В .KID его рисуют так:

- `Graphics.Polygon(точки);`

**Точки** — это массив координат. Каждая точка задаётся парой чисел (x, y). Пример для треугольника:

- Три точки: (100, 50), (50, 150), (150, 150).

Цвет многоугольника задаётся как обычно: `Graphics.Color = "..."` перед вызовом `Polygon`.

Открой и запусти файл `04_PolygonDemo.cs`:

```csharp
using System;
using KID;
using System.Windows;

Console.WriteLine("Треугольник и пятиугольник");

// Треугольник
Graphics.Color = "Tomato";
Graphics.Polygon(new Point[] {
    new Point(150, 40),
    new Point(80, 180),
    new Point(220, 180)
});

// Пятиугольник
Graphics.Color = "SteelBlue";
Graphics.Polygon(new Point[] {
    new Point(150, 60),
    new Point(200, 120),
    new Point(170, 200),
    new Point(130, 200),
    new Point(100, 120)
});
```

На холсте появится красный треугольник и синий пятиугольник. Меняя координаты в массиве точек, можно получать разные формы. Эту тему можно изучать подробнее позже — для начала достаточно знать, что такие фигуры рисуются одной командой.

*В учебнике здесь может быть иллюстрация: пример рисунка из линий и эллипсов.*

### 2.5. Практика: нарисуй программно сам

Теперь можно собрать из изученных фигур целые картинки. Ниже — четыре задания. Каждое можно оформить в отдельный файл и запускать по очереди.

#### Смайлик

Круглое лицо (жёлтый круг), два глаза (маленькие круги или точки), улыбка (полуовал или дуга из линий/эллипса). Цвета — на твой вкус.

Файл: `05_Smiley.cs`

#### Домик

Прямоугольник — стена, треугольник (многоугольник из трёх точек) — крыша, маленький прямоугольник — дверь. Можно добавить квадратное окошко и трубу из прямоугольников.

Файл: `06_House.cs`

#### Машина

Корпус — прямоугольник или скруглённый прямоугольник (можно эллипс + прямоугольник). Колёса — два круга. Окно — прямоугольник другого цвета. Фары — маленькие круги или точки.

Файл: `07_Car.cs`

#### Человечек

Голова — круг, туловище — линия или прямоугольник, руки и ноги — линии. Можно добавить лицо точками или маленькими кругами.

Файл: `08_StickFigure.cs`

Ниже приведены примеры кода для каждого рисунка. Ты можешь скопировать их в свои файлы и потом менять координаты и цвета, чтобы получить свой вариант.

**Пример: смайлик** (`05_Smiley.cs`):

```csharp
using System.Windows;
using KID;

// 1. Жёлтое лицо (большой круг)
Graphics.Color = (255, 255, 0);
Graphics.Circle(150, 150, 100);

// 2. Левый глаз (чёрный кружок)
Graphics.Color = 0x000000;
Graphics.Circle(110, 120, 12);

// 3. Правый глаз (чёрный кружок)
Graphics.Circle(190, 120, 12);

// 4. Красная улыбка (кривая линия)
Graphics.Color = "Red";
var smilePoints = new Point[]
{
	new Point(110, 180),  // Начало улыбки (слева)
	new Point(150, 220),  // Контрольная точка (внизу, создаёт изгиб)
	new Point(190, 180)   // Конец улыбки (справа)
};
Graphics.QuadraticBezier(smilePoints);
```

**Пример: домик** (`06_House.cs`):

```csharp
using System;
using KID;
using System.Windows;

Graphics.Color = "SaddleBrown";
Graphics.Rectangle(100, 180, 200, 120);

Graphics.Color = "DarkRed";
Graphics.Polygon(new Point[] {
    new Point(80, 180),
    new Point(200, 100),
    new Point(320, 180)
});

Graphics.Color = "White";
Graphics.Rectangle(160, 230, 40, 70);

Graphics.Color = "SkyBlue";
Graphics.Rectangle(220, 210, 50, 40);
```

**Пример: машина** (`07_Car.cs`):

```csharp
using System;
using KID;

Graphics.Color = "Crimson";
Graphics.Rectangle(80, 160, 220, 60);
Graphics.Rectangle(120, 120, 140, 50);

Graphics.Color = "LightBlue";
Graphics.Rectangle(140, 130, 50, 35);

Graphics.Color = "Black";
Graphics.Circle(130, 220, 25);
Graphics.Circle(250, 220, 25);

Graphics.Color = "Yellow";
Graphics.Circle(290, 185, 8);
Graphics.Circle(90, 185, 8);
```

**Пример: человечек** (`08_StickFigure.cs`):

```csharp
using System;
using KID;

Graphics.Color = "Black";

Graphics.Circle(150, 50, 25);

Graphics.Line(150, 75, 150, 140);

Graphics.Line(150, 95, 90, 80);
Graphics.Line(150, 95, 210, 80);

Graphics.Line(150, 140, 110, 200);
Graphics.Line(150, 140, 190, 200);

Graphics.Color = "Black";
Graphics.Circle(140, 45, 3);
Graphics.Circle(160, 45, 3);
```

Запускай по одному файлу, смотри результат и экспериментируй с числами и цветами.
