# .KID - C#/.NET IDE для детей

## Обзор проекта

**.KID** — это образовательная интегрированная среда разработки (IDE) для обучения детей программированию на языке C#. Проект разработан специально для начинающих программистов и предоставляет упрощённый интерфейс с визуальным выводом графики и консольным выводом.

### Основная идея

.KID создан для того, чтобы сделать обучение программированию доступным и увлекательным для детей. IDE предоставляет:

- **Простой редактор кода** с подсветкой синтаксиса
- **Визуальный вывод графики** через упрощённый API
- **Консольный вывод** для текстовых программ
- **Многоязычный интерфейс** (русский, английский, украинский)
- **Темы оформления** (светлая и тёмная)

### Целевая аудитория

- Дети и подростки, начинающие изучать программирование
- Преподаватели информатики
- Родители, желающие обучить детей программированию

## Основные возможности

### 1. Редактор кода
- Редактор на базе AvalonEdit с подсветкой синтаксиса C#
- Поддержка отмены/повтора действий (Undo/Redo)
- Автоматическое форматирование

### 2. Выполнение кода
- Компиляция C# кода в реальном времени
- Выполнение скомпилированного кода
- Обработка ошибок компиляции и выполнения
- Возможность остановки выполнения программы

### 3. Графический вывод
- Упрощённый API для рисования фигур (круги, прямоугольники, линии, многоугольники)
- Работа с цветами в различных форматах
- Вывод текста на графическом холсте
- Кривые Безье (квадратичные и кубические)

### 4. Консольный вывод
- Перенаправление `Console.WriteLine` и `Console.Write` в текстовое поле
- Поддержка `Console.ReadLine` для ввода данных
- Очистка консоли

### 5. Интерфейс
- Многоязычная поддержка (ru-RU, en-US, uk-UA)
- Темы оформления (Light, Dark)
- Настраиваемое окно без стандартных рамок Windows
- Разделяемые панели для удобной работы

### 6. Работа с файлами
- Создание нового файла
- Открытие существующих .cs файлов
- Сохранение кода в файл

## Технологический стек

- **.NET 8.0** — платформа разработки
- **WPF** — пользовательский интерфейс
- **AvalonEdit** — редактор кода с подсветкой синтаксиса
- **Microsoft.CodeAnalysis** — компиляция C# кода
- **Microsoft.Extensions.DependencyInjection** — внедрение зависимостей

## Структура проекта

```
KID/
├── KIDLibrary/          # Библиотека для пользовательского кода
│   └── Graphics/        # API для графики
├── Models/              # Модели данных
├── Services/            # Сервисы приложения
│   ├── CodeExecution/  # Выполнение кода
│   ├── Files/          # Работа с файлами
│   ├── Localization/   # Локализация
│   ├── Themes/         # Темы оформления
│   └── DI/             # Dependency Injection
├── ViewModels/         # ViewModels (MVVM)
├── Views/              # Представления (XAML)
├── Resources/          # Ресурсы (строки, иконки)
├── Themes/             # Файлы тем оформления
└── ProjectTemplates/    # Шаблоны проектов
```

## Быстрый старт

1. Откройте проект в Visual Studio
2. Соберите решение (Build Solution)
3. Запустите приложение (F5)
4. Напишите код в редакторе
5. Нажмите "▶️ Запустить" для выполнения

### Пример кода

```csharp
using System;
using KID;

// Приветствие в консоли
Console.WriteLine("Привет, Мир!");

// Рисуем круг
Graphics.Color = "Blue";
Graphics.Circle(150, 150, 50);
```
